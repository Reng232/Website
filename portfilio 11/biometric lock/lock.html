Biometric DoorLock

Introducing a cutting-edge biometric door lock system. This project showcases the ingenuity of its creators, utilizing the Arduino IDE alongside a fingerprint scanner to ensure only authorized users gain access. Dive deeper into the details of this impressive innovation as you journey through its functionalities.

Who We Are

Jogita and Emma chose to build this biometric door lock system for their project because it offers a guaranteed method of keeping unauthorized users outside. In a real-world setting, such a system can protect against trespassing or unauthorized access to crucial information. Furthermore, it's more secure than a traditional key, which can be lost or stolen, as a fingerprint is unique to a singular individual. Adding another layer of verification, the phone password ensures security even if fingerprint verification is compromised. Inspired by a social media video depicting a guy using a fingerprint on a door knob, Jogita and Emma embarked on this project, utilizing their ingenuity to create a modern solution for enhanced security.

Key Features

Two key features of this project are the fingerprint sensor and the OLED display. The fingerprint sensor detects a person's unique fingerprint, while the OLED display indicates whether the door is locked or unlocked. Additionally, the OLED display alerts users if a fingerprint is not registered properly. These features contribute to the project's functionality and enhance its security measures.

Materials

The materials that were used in this project were the following:

  • Esp 32
  • R3o7 optical fingerprint sensor
  • OLed
  • 12v solenoid lock
  • Single channel 5v relay
  • breadboard
  • buzzer

Obstacles Faced

Some obstacles faced during this project included the wiring,
which was challenging because many components were small and required precise connections.
Additionally, ensuring the voltage was correct was crucial to prevent any explosions.
There was a small bug in the code that affected the wiring.
Lastly, the 3D-printed parts were not the right size
